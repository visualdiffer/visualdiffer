### How to debug

> [!NOTE]
> the application must be installed in the `/Applications` folder or the debugger won't work
> the VisualDiffer scheme is configured to create the symbolic link inside `/Applications` by running a `Run Script`


1. Select `FinderCompare` Scheme
2. Run
3. Choose `Finder` from the "app to run" picker
4. Open a `Finder` window and select the extension menu
